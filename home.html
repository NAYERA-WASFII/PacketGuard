Process

How PacketGuard
works

01
Network Discovery
ARP scans discover every active device on the subnet, including sleeping phones and IoT devices.
02
Packet Capture
Scapy captures live packets and feeds them through rule-based and ML-based detection pipelines.
03
Threat Analysis
Each packet is scored by 4 detectors: Port Scan, SYN Flood, High Rate, and Isolation Forest.
04
Alert & Visualize
Confirmed threats are logged, classified by severity, and pushed to the dashboard as notifications.

About

Built as a graduation project in network security

PacketGuard combines classical intrusion detection with modern ML to deliver a complete monitoring solution.

Real-time packet analysis using Python and Scapy for deep network visibility
Isolation Forest ML model trained on network traffic to detect anomalies
Flask-powered REST API serving a live updating web dashboard
Supports all device types — phones, laptops, smart devices, routers
Runs on Windows with administrator privileges for full packet capture
